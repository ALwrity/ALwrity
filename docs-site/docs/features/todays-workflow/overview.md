# Today's Workflow Overview

The **Today's Workflow** is the central command center of ALwrity — the single place where end users interact with all backend intelligence. It transforms complex AI analysis, competitor monitoring, SEO insights, and content strategy into a curated, actionable daily task list.

Every morning (or on demand), ALwrity generates a personalized set of tasks across the **6 Content Lifecycle Pillars**, giving users a clear, prioritized plan for the day.

## Why It Matters

- **Single Pane of Glass**: Instead of jumping between dashboards, users get one unified daily plan
- **AI-Powered Prioritization**: Tasks are generated by a committee of specialized AI agents, not static rules
- **Action-Oriented**: Every task has a direct action — navigate, generate, review — with one-click execution
- **Self-Learning**: The system tracks which tasks you complete or dismiss, and adapts future suggestions

## The 6 Content Lifecycle Pillars

| Pillar | Focus | Typical Task |
|--------|-------|-------------|
| **Plan** 🎯 | Content strategy & planning | Review today's content plan, identify gaps |
| **Generate** ✍️ | Content creation | Write a blog post, create a LinkedIn update |
| **Publish** 🚀 | Content distribution | Schedule or publish today's content |
| **Analyze** 📊 | Performance analysis | Review SEO dashboard, check semantic health |
| **Engage** 💬 | Social engagement | Engage on LinkedIn, respond to comments |
| **Remarket** 🔄 | Content repurposing | Repurpose a blog into social posts |

## Key Features

### AI Agent Committee
6 specialized agents propose tasks daily:
- **Content Strategy Agent** — Identifies content opportunities
- **Strategy Architect Agent** — Suggests strategic pivots from SIF semantic analysis
- **SEO Optimization Agent** — Flags SEO issues and opportunities
- **Social Amplification Agent** — Recommends engagement actions
- **Competitor Response Agent** — Alerts on competitor moves
- **Content Gap Radar Agent** — Scores content opportunities from SERP data, competitor deep-dives, and trend momentum

### Self-Learning Memory
The system remembers which tasks you dismiss and avoids suggesting similar ones for 7 days using txtai semantic search.

### Pillar Coverage Guarantee
Every pillar gets at least one task per day, ensuring balanced progress across the full content lifecycle.

### Contextuality Validation
Each task is validated against your actual onboarding data and recent agent alerts, ensuring nothing feels generic.

### Committee Watchdog Audit
After the committee generates proposals, the **ContentGuardianAgent** independently audits the output — scoring each agent's reasoning quality, detecting coverage gaps across pillars, flagging duplicate proposals, and generating alerts for serious faults. The audit runs automatically every cycle and is surfaced on the Team Activity page (see below).

### Trend Signals
The **Trend Surfer Agent** monitors Google Trends and market signals after the workflow is generated, logging the top 5 emerging opportunities with impact scores, urgency levels, and suggested content angles — visible on the Team Activity page.

## How It's Surfaced

### Main Dashboard
Today's Workflow is the first thing users see on the main dashboard:

```
MainDashboard
├── DashboardHeader (with Start/Resume button)
├── ContentLifecyclePillars (6 gradient pillar cards)
│   └── Each card → EnhancedTodayChip (shaking "Today" pill)
│       └── EnhancedTodayModal (task details, complete/skip actions)
├── WorkflowProgressBar (if workflow is active)
└── WorkflowHeroSection (first visit / workflow not started)
```

When you open the dashboard, you'll see 6 pillar cards. Click the "Today" chip on any pillar to see its tasks. The progress bar at the top tracks your overall completion.

### Team Activity Page
The **Team Activity** page gives you a real-time view of what the agents are doing behind the scenes:

- **Real-time Feed** — Every agent event (proposals, audits, errors, initialisations) streams to a live chronological feed
- **Committee Summary** — Recap of today's committee meeting: which agents proposed how many tasks, acceptance rates
- **Quality Audit Panel** — ContentGuardianAgent's audit report: committee health gauge (0–100), per-agent critique cards with score bars and issue details, coverage gaps, overlaps
- **Trend Signals Panel** — Trend Surfer Agent's top 5 opportunities with urgency chips, impact scores, and content angle suggestions
- **Agent Status Panel** — Live status of each registered agent (active, error, disabled)
- **Alert Banner** — Persistent alerts for serious findings, dismissible with server-side persistence
